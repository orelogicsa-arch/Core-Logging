Usuarios del Sistema

Flujo de Trabajo

  • 1. Login — Seleccionar usuario e ingresar contraseña.
  • 2. Crear / seleccionar sondaje — Barra de sondaje activo.
  • 3. Header — Completar metadatos. Ingresar la Profundidad Final antes de registrar intervalos.
  • 4. Marcación — Recuperación física por cajas. From se llena automáticamente desde el To anterior. ⚡ Relleno automático para sondajes estándar.
  • 5. Geología — Descripción litológica con Ore Zone, color, contacto, granulometría, fósiles y flags Dens/Muestrea.
  • 6. Muestreo — Importar desde Geología, insertar duplicados y controles, auto-numerar y generar plan de muestreo.
  • 7. Densidad — Importar intervalos DENS=Y, completar medidas. Validación automática de cada fila.
  • 8. Exportar — Genera XLSX con las secciones seleccionadas, una hoja por sección.

Simbología de Ore Zones

Reglas de Validación — Densidad

  • ✗ Error — Peso Húmedo < Peso Seco (crítico)
  • ⚠ Aviso — Diferencia entre pesos > 20%
  • ⚠ Aviso — CV de L1–L4 o D1–D4 entre 5% y 10%
  • ✗ Error — CV > 10%

Protocolo QAQC Muestreo

  • PRI — Muestra principal del intervalo
  • TWS — Segunda mitad del core (twin)
  • DUP — Duplicado de campo por cuarteo
  • CDP — Duplicado de preparación
  • PDS — Duplicado de pulpa
  • CBK — Blanco de campo (sin intervalo)
  • CRM — Material de referencia certificado (BAY01-19, BAY02-19, BAY03-19)

Persistencia Local

Todos los datos se guardan en localStorage del navegador. No se requiere servidor. Cambiar de sondaje dispara guardado automático. Use Exportar para generar respaldos XLSX.
