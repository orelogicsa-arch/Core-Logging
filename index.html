📖 Guía de Uso — Core Logging OreLogic

🔐 Inicio de Sesión

Selecciona tu usuario del dropdown y escribe tu contraseña. El administrador (usuario admin, contraseña 1234) tiene acceso completo. Los demás usuarios acceden como Geólogo/Técnico.

📋 Flujo de Trabajo Recomendado

  1. Crear sondaje → Botón "+ Nuevo" en la barra superior. Asigna ID, proyecto, tipo y diámetro.
  2. Header → Completa azimuth, dip, profundidad final, geólogo, fechas y coordenadas collar. Guarda con 💾 Guardar Header.
  3. Marcación → Usa ⚡ Rellenar automático (cajas de 1.5m) o agrega filas manualmente. Los intervalos se validan automáticamente (sin gaps, sin solapamientos).
  4. Geología → Registra litología, ore zone, color, intensidad, contacto, granulometría por intervalo. La columna Simbología muestra el código gráfico de lito y ore zone. Marca Se Muestrea? = Y en los intervalos a muestrear.
  5. Muestreo → Usa 📥 Importar de Geología para traer automáticamente los intervalos marcados con Y. Luego auto-numera las muestras. Inserta duplicados y controles con los botones correspondientes.
  6. Densidad → Registra pesos húmedo/seco y las 4 medidas de largo (L) y diámetro (D). El sistema valida automáticamente consistencia de medidas y calcula densidad.
  7. Exportar → Botón ⬇ Exportar ZIP en el topbar. Selecciona uno o varios sondajes y las secciones a incluir. Se descarga un archivo ZIP con CSV por carpeta.

⚖️ Densidad — Validaciones

  • Peso Húmedo > Peso Seco: el peso húmedo siempre debe ser mayor. Una diferencia > 20% genera advertencia.
  • Medidas L1–L4: representan largos del core en cm. El coeficiente de variación (CV) debe ser < 5% para ser válido (< 10% genera advertencia).
  • Medidas D1–D4: diámetros del core. Mismo criterio CV < 5%.
  • El Volumen se calcula como cilindro: V = π × (D̄/2)² × L̄
  • La Densidad = Peso Seco / Volumen (g/cm³).

🔀 Controles de Calidad (QAQC)

  • Duplicado (DUP): copia del mismo intervalo de una muestra PRI. Se inserta inmediatamente debajo de la muestra seleccionada.
  • Twin (TWS): segunda mitad del core del mismo intervalo.
  • Blanco (CBK): material estéril certificado para detectar contaminación cruzada.
  • CRM/Estándar: material de referencia certificado (BAY01-19, BAY02-19, etc.) para verificar precisión analítica.
  • Los controles se insertan siempre debajo del intervalo referenciado.

⚙️ Administración (solo Admin)

  • Listas configurables: todos los desplegables (litología, ore zone, geólogos, etc.) se editan desde Admin → Listas.
  • Usuarios: crear, activar/desactivar, resetear contraseñas desde Admin → Usuarios.
  • Log de actividad: registro completo de todas las acciones con timestamp, usuario y sondaje.
ℹ️ Persistencia: todos los datos se guardan en el localStorage del navegador. Para compartir entre equipos, usa la exportación ZIP e importa los CSV al sistema corporativo. Versión 3.0 · OreLogic S.A. · Miski Mayo — Bayóvar.
